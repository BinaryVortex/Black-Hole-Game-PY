# 🕳️ Black Hole Game (Python)

## 📌 Overview  
This repository contains a **Black Hole Game (board-style game) built using Python programming language**.  
The game is designed as a strategic and interactive experience where players navigate challenges influenced by a “black hole” mechanic.

---

## 🎯 Purpose  
- Practice Python game development  
- Improve logic building and decision-making skills  
- Learn turn-based / board game mechanics  
- Enhance problem-solving and algorithmic thinking  
- Build an interactive console-based game system

## 🧠 Key Features  
- 🎮 Turn-Based Gameplay  
- 🕳️ Black Hole Mechanic (special game effect)  
- 👥 Player Interaction System
